## The Issue

## Solution (2 steps - takes 1 minute)

### Step 1: Go to Repository Settings
1. Open: https://github.com/zhirkoalexander-maker/nukakola
2. Click **Settings** (top menu bar)
3. In left sidebar, click **Pages** (under Deployment section)

### Step 2: Enable GitHub Pages
In the Pages settings:
1. Under "Source", select: **Deploy from a branch**
2. Branch dropdown: Select **main**
3. Folder dropdown: Select **/docs**
4. Click **Save**
