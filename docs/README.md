# Base Defense - Game Website

This directory contains the GitHub Pages website for the Base Defense tower strategy game.

## Website URL

The website is available at: **https://zhirkoalexander-maker.github.io/nukakola/**

## Contents

- `index.html` - Main game documentation and info page with:
  - Game overview and features
  - Gameplay modes (Classic, Hard, Survival)
  - Controls and keybindings
  - Building materials list
  - Installation instructions
  - Strategy tips

## How GitHub Pages Works

This repository uses GitHub Pages to automatically host the `docs/index.html` file as a website. Any changes pushed to the `main` branch will be automatically deployed.

## Customizing the Website

To modify the website:
1. Edit `docs/index.html`
2. Commit and push changes
3. Wait 1-2 minutes for GitHub to rebuild the site
4. Visit the URL above to see your changes

## Features

- Responsive design that works on mobile and desktop
- Dark theme optimized for gaming communities
- Comprehensive game documentation
- Direct links to GitHub repository and downloads
- Installation and setup guide
