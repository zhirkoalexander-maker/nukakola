‚≠ê Strategy Tips

  • Defense First: Build defensive walls early to protect your base core
  • Unit Synergy: Mix different unit types (Archer for range, Warrior for melee, Mage for support)
  • Weapon Choice: Switch weapons based on enemy type. Sniper for single targets, Hammer for groups
  • Money Management: Save money for emergencies but don't leave it unused
  • Positioning: Keep your base core away from enemies. Build walls in strategic spots
  • Wave Timing: Hire units between waves for maximum efficiency
  • Learning Curve: Start with Classic mode to master the mechanics

‚ö° Quick Start Installation

Step 1: Install Python

  • Windows/Mac/Linux: Download from python.org (version 3.8 or higher)
  • During installation, CHECK "Add Python to PATH"
  • Verify: Open terminal/command prompt and type: python --version

Step 2: Clone Repository

  1. Open terminal/command prompt in desired folder
  2. Run: git clone https://github.com/zhirkoalexander-maker/nukakola.git
  3. Navigate: cd nukakola

Step 3: Install Pygame

Run in terminal/command prompt:

pip install pygame

Step 4: Run the Game

Run in terminal/command prompt:

python building_game.py
